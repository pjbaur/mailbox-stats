## Project Overview

This is a Gmail statistics tool that analyzes mailbox data to help clean up Gmail accounts. The tool focuses on identifying:
- Senders with the most emails
- Senders with the largest emails (including attachments)

## Tech Stack

- **Python**: 3.12
- **Cloud Platform**: Google Cloud Platform (GCP)
- **API**: Gmail API

## Project Status

This is an early-stage project. The codebase structure and implementation details will be defined as development progresses.
